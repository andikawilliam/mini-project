# Invoice Frontend

## How to use it

Requires:
Node JS (>= 12.x)

**Step 1** - Clone the project

```bash
$ git clone "this repo"
$ cd invoice-frontend
```

<br >

**Step 2** - Install dependencies via NPM or yarn

```bash
$ npm i
```

<br />

**Step 3** - Start in development mode

```bash
$ npm run start 
```
